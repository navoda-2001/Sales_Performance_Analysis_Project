# 📊 Sales Performance Analysis Project

## 📌 Overview
This project focuses on analyzing monthly sales data to identify key business trends, top-performing regions, and product performance.  
The analysis was carried out using **Python (Pandas)** for data processing and **Power BI** for dashboard visualization.

The objective of this project is to support data-driven decision-making by transforming raw sales data into meaningful business insights.

---

## 🛠️ Tools & Technologies Used
- **Python**
- **Pandas**
- **Matplotlib**
- **Microsoft Power BI**
- **Excel**
- **GitHub**

---

## 📂 Dataset Details
The dataset contains the following columns:

- **Order_ID** – Unique order number
- **Month** – Sales month
- **Region** – Sales region
- **Product** – Product name
- **Quantity** – Number of items sold
- **Unit_Price** – Price per item
- **Sales** – Total sales amount

---

## 📈 Analysis Performed
The following analysis tasks were completed:

- Total sales calculation
- Average sales analysis
- Highest and lowest sales identification
- Monthly sales trend analysis
- Region-wise sales performance
- Product performance comparison
- Data visualization using charts
- Interactive dashboard creation in Power BI

---

## 📊 Dashboard Features
The Power BI dashboard includes:

- **Total Sales KPI Card**
- **Average Sales KPI Card**
- **Monthly Sales Trend (Bar Chart)**
- **Sales by Region (Pie Chart)**
- **Product Performance Chart**
- **Month Slicer for Filtering**

---

## 💡 Key Insights
- Colombo generated the highest total sales revenue.
- Sales increased steadily from **January to March**.
- Laptop sales outperformed phone sales.
- March was the best-performing sales month.
